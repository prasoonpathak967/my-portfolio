# 🌐 Personal Portfolio Website

This is my personal developer portfolio website where I showcase my projects, skills, and experience as a Full Stack Developer. The portfolio highlights my work, technical expertise, and provides a way for recruiters or collaborators to connect with me.

🔗 **Live Demo:**  
https://prasoon-folio-spark.vercel.app/

---

# 🚀 Features

- Modern and responsive UI
- Personal introduction and profile
- Project showcase section
- Skills and technologies section
- Contact information
- Clean and professional design
- Fully responsive across devices

---

# 🛠 Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS
- Vite

Tools & Deployment:
- Vercel
- Git & GitHub
